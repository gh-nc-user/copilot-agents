  ✅ WORKFLOW ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════════

1. User triggers DraftMemo via recognized intent
   - Trigger phrases: "Draft memo", "Create memo", "Write investment memo"

2. DraftMemo collects:
   - Company name & sector
   - Market Analysis Agent output (with Confidence & Evidence)
   - Competitive Landscape Agent output (with Confidence & Evidence)

3. ValidateInputs validates:
   - Checks both outputs contain "Confidence" section
   - Checks both outputs contain "Evidence" section
   - If invalid: error message + EndDialog
   - If valid: continue to MemoSections

4. MemoSections orchestrates 5 section generators:
   ├── CompanyOverviewSection   → Topic.CompanySection
   ├── MarketAnalysisSection    → Topic.MarketSection (verbatim)
   ├── CompetitiveLandscapeSection → Topic.CompetitiveSection (verbatim)
   ├── KeyRisksSection          → Topic.RisksSection
   └── OpenQuestionsSection     → Topic.OpenQuestionsSection

5. CompileFinalMemo assembles:
   - Concatenates all 5 sections
   - Adds metadata (date, company, sector)
   - Displays complete memo
   - Offers export/save options
