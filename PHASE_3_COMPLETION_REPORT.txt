INTAKE PHASE:
  IntakeIdentify topic captures:
  - Topic.CompanyName
  - Topic.Sector
  - Topic.Stage
  - Topic.DocumentDescription
  - Topic.IntakeSummary (generated)

  CallMarketAgent calls Market Analysis Agent with:
  - CompanyName, Sector, Stage, DocumentDescription
  Returns and stores:
  - Topic.MarketAgentResult → Topic.MarketAgentFindings (unchanged)
  - Detects: Topic.MarketHasConfidenceSection

COMPETITIVE ANALYSIS PHASE:
  CallCompetitiveAgent calls Competitive Landscape Agent with:
  - CompanyName, Sector, Stage, DocumentDescription
  Returns and stores:
  - Topic.CompetitiveAgentResult → Topic.CompetitiveAgentFindings (unchanged)
  - Detects: Topic.CompetitiveHasConfidenceSection

VALIDATION PHASE:
  ValidateOutputs checks:
  1. Both outputs present (not blank)
  2. Both confidence sections present
  3. Low confidence indicators detected
  4. Routes based on validation results:
     - ✓ Valid → Memo Writer
     - ⚠️ Incomplete → User decision
     - ❌ Missing → Escalation

MEMO GENERATION PHASE:
  CallMemoWriter calls Deal Memo Writer Agent with:
  - CompanyName, Sector, Stage
  - MarketFindings (unchanged from Market Agent)
  - CompetitiveFindings (unchanged from Competitive Agent)
  - IntakeSummary
  Returns:
  - Topic.DealMemoResult → Topic.DealMemo (unchanged)
  - Includes Confidence & Evidence from both agents
  - Displays IC review flag if low confidence detected

OUTPUT PHASE:
  User receives:
  - Professional Deal Memo with both analyses
  - IC review flag if applicable
  - Next steps and options for follow-up

================================================================================
ERROR HANDLING IMPLEMENTATION
================================================================================

ERROR SCENARIO 1: Agent Timeout/Failure
  Detection: IsBlank(Topic.MarketAgentResult) || IsBlank(Topic.MarketAgentFindings)
  Handling: Retry once, then escalate
  User Message: "⚠️ Agent did not return complete response... (2 attempts)"
  Escalation: Routes to Escalate topic if both fail

ERROR SCENARIO 2: Missing Confidence & Evidence
  Detection: Find(output, "Confidence & Evidence") = 0
  Handling: Flag output, ask user to accept or request completion
  User Message: "⚠️ Analysis missing Confidence & Evidence section..."
  Escalation: Optional user choice

ERROR SCENARIO 3: Low Confidence Detection
  Detection: Find(Lower(output), "low") > 0
  Handling: Alert user, flag for review
  User Message: "⚠️ Analysis indicates LOW CONFIDENCE in findings..."
  Escalation: Sets FlagForICReview if both agents report low confidence

ERROR SCENARIO 4: Memo Generation Failure
  Detection: IsBlank(Topic.DealMemo) after agent call
  Handling: Retry once, then escalate
  User Message: "❌ Memo generation failed... attempting retry..."
  Escalation: Routes to Escalate topic if both fail

ERROR SCENARIO 5: Critical Data Missing
  Detection: Either market or competitive output is blank after retries
  Handling: Immediate escalation
  User Message: "❌ CRITICAL: One or both specialist analyses are missing..."
  Escalation: Routes to Escalate topic immediately
