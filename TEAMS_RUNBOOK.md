# 📘 Deal Analysis Agent System - Teams User Guide

**Quick Reference Guide for End-Users**

> **Not a tech person?** No problem! This guide walks you through using the AI agents step-by-step.

---

## 🎯 What This System Does

The **Deal Analysis Agent System** automatically analyzes business deals for you:

1. 📊 **Extracts deal information** from your documents (Who? How much? What industry?)
2. 🌍 **Analyzes market opportunity** (Is this market growing? What's the potential?)
3. 🏆 **Analyzes competitive landscape** (Who are the competitors? What's our advantage?)
4. ✍️ **Writes an analysis memo** (5-section professional summary with recommendation)

**Time to complete**: 15-30 seconds per deal

**Format**: AI-generated memo delivered directly in Teams

---

## 🚀 How to Get Started

### Step 1: Open Copilot in Teams

1. Open **Microsoft Teams**
2. Click the **Copilot** icon in the left sidebar
   - Look for the 🔮 icon
   - Or search for "Copilot" in the search bar
3. You'll see the Copilot interface

### Step 2: Find the Deal Analyzer

1. Look for the **"Agents"** tab in Copilot
2. You'll see a list of available AI agents
3. Find and click **"Deal Orchestrator"**
   - This is the main entry point
   - It coordinates all the analysis
4. A new chat window opens

---

## 📋 How to Use: Step-by-Step

### Option A: Upload a Deal Document

**Best for**: PowerPoint decks, PDFs, Word documents

1. In the chat window, click the **attachment icon** (📎 or 📤)
2. Select a deal document from your computer (PPT, PDF, DOCX, Excel)
3. Type in the chat: `"Please analyze this deal for me"` or `"What's your assessment?"`
4. Click **Send** (or press Enter)
5. ⏳ Wait 15-30 seconds for analysis
6. The memo appears in the chat

### Option B: Describe the Deal in Chat

**Best for**: Quick analysis without uploading files

1. In the chat window, type a description like:

   ```
   I have a deal to analyze.
   - Company: TechStartup Inc
   - Deal size: $50 million
   - Industry: Enterprise SaaS
   - Main competitors: Salesforce, ServiceNow
   - Growth rate: 40% YoY
   - Key customers: Fortune 500 companies
   ```

2. Click **Send** (or press Enter)
3. ⏳ Wait 15-30 seconds
4. The memo appears in the chat

### Option C: Ask Questions About a Deal

**Best for**: Specific questions about deals you've already analyzed

After the memo is generated, you can ask follow-up questions:

- "What's the biggest risk in this deal?"
- "Can we negotiate on valuation?"
- "What's the market timeline?"

The agent will answer based on the analysis.

## 📄 What You'll Get: The Analysis Memo

The agent generates a **5-section professional memo**:

### Section 1: Executive Summary
- Quick overview of the deal
- Key metrics at a glance
- Overall recommendation (Buy/Pass/Hold)

### Section 2: Deal Overview & Valuation
- Company name and description
- Deal structure
- Valuation analysis
- Price per share (if applicable)

### Section 3: Market Opportunity
- Total addressable market (TAM)
- Market growth rate
- Target customer segments
- Market confidence level (High/Medium/Low)

### Section 4: Competitive Landscape
- Key competitors
- Competitive advantages
- Market positioning
- Competitive confidence level (High/Medium/Low)

### Section 5: Investment Recommendation
- Detailed recommendation (Buy/Hold/Pass)
- Key reasons for recommendation
- Top risks and opportunities
- Final confidence score

**Example Memo Structure:**
```
DEAL ANALYSIS MEMO
==================

Executive Summary
- Company: AcmeTech Inc
- Deal Value: $75M
- Recommendation: BUY (High confidence)

Deal Overview
- Industry: Enterprise SaaS
- Revenue: $15M ARR
- Growth: 45% YoY

Market Opportunity
- TAM: $5.2B
- Market Growth: 22% CAGR
- Confidence: HIGH

Competitive Landscape
- Competitors: CompanyA, CompanyB, CompanyC
- Differentiation: 2x faster, 30% cheaper
- Confidence: MEDIUM

Recommendation
- STRONG BUY
- Confidence: HIGH
- Rationale: Strong market, differentiated product...
- Risks: Slower sales cycles, concentration risk...
- Opportunities: International expansion, product upsell...
```

## ❓ Frequently Asked Questions

### Q1: What if the agent says "Low Confidence"?

**Answer**: This usually means:
- ✗ Not enough information in the deal materials
- ✗ Market data is incomplete or unclear
- ✗ Competitive information is hard to find

**What to do**:
- Provide more details about the deal
- Upload additional research documents
- Ask the agent for specific information it needs
- Run analysis again with more data

---

### Q2: What if the agent times out or doesn't respond?

**Answer**: Timeouts are rare but can happen if:
- 🌐 Your internet connection is slow
- 🔄 Power Platform is experiencing delays (very rare)
- 📄 The documents are extremely large

**What to do**:
1. Wait 30 seconds and check if message appears
2. Try again with a simpler query
3. Upload documents one at a time instead of all together
4. If it persists, contact [Support Contact]

---

### Q3: Can I run analysis on the same deal multiple times?

**Answer**: **Yes!** Absolutely.

- Run it once with initial data
- Run it again after you get more information
- Run it again before final decision
- Compare memos to see how confidence changes

**Tip**: Save each memo with a timestamp so you can track how your analysis evolves.

---

### Q4: What if I don't trust the recommendation?

**Answer**: This is normal! The agent is a tool, not a decision-maker.

**Always verify**:
- ✓ Is the market analysis correct? (Cross-check with your research)
- ✓ Are competitors identified correctly? (Validate your competitive intel)
- ✓ Is the valuation fair? (Compare to recent comps)
- ✓ Are risks identified? (What else should we consider?)

**Use the memo as input** to your decision-making process, not as the final answer.

---

### Q5: Where do I save the memo?

**Answer**: Multiple options:

**Option 1: Copy to Word/PowerPoint**
1. Click on the memo text in Teams
2. Select all (Ctrl+A / Cmd+A)
3. Copy (Ctrl+C / Cmd+C)
4. Paste into Word or PowerPoint

**Option 2: Take a Screenshot**
1. Screenshot the memo in Teams
2. Save as PNG/PDF

**Option 3: Forward in Teams**
1. Right-click the memo message
2. Select "Forward"
3. Send to your email or Teams channel

**Option 4: Print**
1. Take screenshot or copy to Word
2. Print to PDF or physical printer

---

### Q6: Can I analyze multiple deals at once?

**Answer**: **Not at the same time**, but you can run multiple analyses back-to-back.

1. Analyze Deal #1 → Get Memo #1
2. Start new chat conversation
3. Analyze Deal #2 → Get Memo #2
4. Compare the two memos

**Tip**: Keep them in separate Teams conversations so you can reference them side-by-side.

---

### Q7: What information does the agent need?

**Minimum information** for a good analysis:
- ✓ Company name or description
- ✓ Deal size (price, valuation, terms)
- ✓ Industry
- ✓ Customer profile
- ✓ Revenue/metrics (if available)

**Extra information** makes analysis better:
- ✓ Market research
- ✓ Competitive intelligence
- ✓ Financial projections
- ✓ Management team background

---

### Q8: Is this secure? What about confidential information?

**Answer**: Security depends on your organization's setup.

**Check with your admin about**:
- ✓ Data residency (where information is stored)
- ✓ Compliance requirements (GDPR, SOC2, etc.)
- ✓ Data retention policies (how long data is kept)
- ✓ Who can see analysis memos

**Best Practice**: Don't paste sensitive data into chat if you're unsure. Contact your IT team if you have security concerns.

---

### Q9: Can the agent make decisions for me?

**Answer**: **No.** The agent provides analysis, you make decisions.

The agent:
- ✓ Provides data-driven analysis
- ✓ Identifies opportunities and risks
- ✓ Summarizes competitive landscape
- ✓ Makes a recommendation

You should:
- ✓ Review the analysis
- ✓ Validate the findings
- ✓ Consider internal strategy
- ✓ Make the final decision
- ✓ Take responsibility for the decision

---

4. If the format could be better

Contact [Support Contact] with:

## 🎓 Tips & Best Practices

### Tip 1: Upload Clean Documents
- **Good**: Deal deck PDF (clean, organized)
- **Better**: Deck + financial model + market research
- **Avoid**: Scanned images with poor quality

**Tip**: If documents are messy, describe the deal in chat instead.

---

### Tip 2: Be Specific About What You Need
Instead of: `"Analyze this deal"`

Try: `"Analyze this deal and tell me if we should acquire this company or pass"`

---

### Tip 3: Run Analysis Early and Often
- Don't wait for perfect information
- Run analysis with initial data
- Run again as you learn more
- Track how confidence levels change
- Use this to identify gaps in your research

---

### Tip 4: Combine with Your Own Research
The agent is ONE INPUT into your decision process.

Always also:
- ✓ Interview management team
- ✓ Validate financial projections
- ✓ Visit customer references
- ✓ Conduct technical due diligence
- ✓ Get legal review

---

### Tip 5: Export Memos for Records
Create a **deal analysis folder** to store memos:
- Folder: `Deals/2024/CloudTech_Analysis/`
- Files: `CloudTech_Memo_v1.docx`, `CloudTech_Memo_v2.docx`
- This helps with audit trail and decision documentation
